# Honey

Honey is currently a simple and sweet async runtime for learning purposes.

## Building

### Dependencies
```
cargo
rustc
```
You can use `nix-shell` if using Nix to get the dependencies.
```
nix-shell
```

To build:
```
cargo build
```
